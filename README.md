## Run Locally

Clone the project

```bash
 https://github.com/haarsh24/quiz-app.git
```

Go to the project directory

```bash
  cd quiz-app
```

Open projects

```bash
    index.html
```

Run code.

## Contains

- Homepage
- Category page
- Rules page
- Questions page
- Result page
- Authentication page

## Technologies used

- ReactJS
- [LIT UI](https://litui.netlify.app/) component library
- HTML, CSS, JavaScript
- [Mockbee](https://mockbee.netlify.app/) for backend
- **Hosted:** Netlify
